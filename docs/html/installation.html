Installation¶

Requirements¶

  • Python ≥ 3.7

  • git

  • Chromium or Google Chrome browser, and ChromeDriver

Installation¶

Simply clone the repo and install the Python package.

git clone git@github.com:nrsyed/proboards-scraper.git
cd proboards-scraper
pip install .
